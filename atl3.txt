#Steps To Complete Experiment No. 3 :- 

### **1. Install Ruby**  
- Download **Ruby+Devkit 3.0.1** (or later) from:  
[https://rubyinstaller.org/downloads/].
- Run the installer, check **"Add Ruby to PATH"**, and install the **MSYS2 development toolchain** when prompted.  

### **2. Install Rails**  
```cmd
gem install rails -v 6.1.3.2
```

### **3. Install Node.js & Yarn (Required for Rails 6+)**  
- Download Node.js (LTS) from: [https://nodejs.org/](https://nodejs.org/)  
- Install Yarn via:  
```cmd
npm install -g yarn
```
